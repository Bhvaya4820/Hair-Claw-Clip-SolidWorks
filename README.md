# Hair Claw Clip Design and Assembly using SolidWorks

## Project Overview

This project involves the complete design and assembly of a plastic hair claw clip using SolidWorks. The model consists of multiple components assembled to create a functional opening and closing mechanism. The project focuses on CAD modeling, assembly design, and Design for Manufacturing (DFM) principles.

---

## Software Used

- SolidWorks 2021

---

## Components

- Upper Jaw
- Lower Jaw
- Hinge Pin
- Spring

---

## Project Features

- Parametric 3D modeling
- Multi-part assembly
- Functional hinge mechanism
- Spring integration
- Motion verification
- Design for Manufacturing (DFM)

---

## Design Process

1. Designed individual components in SolidWorks.
2. Created the upper and lower jaws.
3. Modeled the hinge pin and spring.
4. Applied assembly mates to constrain all components.
5. Verified the opening and closing motion of the clip.
6. Improved the model using DFM guidelines such as fillets, draft angles, and uniform wall thickness.
